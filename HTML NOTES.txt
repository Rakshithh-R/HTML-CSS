----------------------------------------------HTML-------------------------------------------------------
-> Hyper-Text Markup Language [HTML].
-> Tool used to create website. 
-> It gives instructions to a computer, where computer follows the instructions to create our website. 


---|SYNTAX|---
~ Basically syntax are rules how a coding language like html should be written so that computer understands.
Example: 
<button>Hello</button>
<p>Nice job...!!</p>
- Here <button>, <p>, </p> and </button> are html tags.
- HTML tags basically tell computer what we are trying to create.
- <p> and <button> are Opening tags, </button> and </p> are Closing tags.


---|ELEMENTS|---
~ Elements is just a generic term that we use to describe anything that we can display on the web page.
Example: 
<button>Hello</button> 
- Starting from opening tag till the end tag we call it as elements.


---|HTML TAGS|---
1. <p> -> Paragraph tag.
       -> To create texts in paragraph.

2. <button> -> Button tag.
            -> To create buttons.
            
3. <a> -> Anchor tag.
       -> It links to another websites.
       -> Example: <a href="https://www.youtube.com">Link to Youtube</a>

       ---|HREF ATTRIBUTE|---
       -> Here, href is as HTML ATTRIBUTE, ATTRIBUTES basically modifies how an element behaves.
       -> Left side(href) is an attribute name and right side("https://www.youtube.com") is attribute value.

       ---|TARGET ATTRIBUTE|---
       -> Here in anchor tag we can also use TARGET ATTRIBUTE.
       -> Example: <a href="https://www.youtube.com" target="_blank">Link to Youtube</a>
       -> Here, TARGET ATTRIBUTE determines whether the link opens in the current page or in the new tab.
       -> By default if we dont have target attribute link opens in current page.
